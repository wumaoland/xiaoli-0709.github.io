C++学习记录（一）

发表于2022-07-13|更新于2022-07-14
|阅读量:

一、开头

本篇用于记录自己学习C++ 中一些容易忘记的知识。

这篇文章的主要内容为数组vector 容器 和 字符串。

二、字符串

C++ 相较于C 语言而言，对字符串处理新添了很多的函数。对于一个习惯了C语言编程思想的我而言，老想着自己写函数去实现那些过程，结果写来写去还写的一团糟，于是乎将这些函数给记录下来。

首先是引入头文件，为 和

  • cstring
  • string

string类提供了一系列针对字符串的操作，比如:

    1. append() -- 在字符串的末尾添加字符
    1. find() -- 在字符串中查找字符串
    1. insert() -- 插入字符
    1. length() -- 返回字符串的长度
    1. replace() -- 替换字符串
    1. substr() -- 返回某个子字符串

三、Vector 容器

向量（Vector）是一个封装了动态大小数组的顺序容器（Sequence Container）。跟任意其它类型容器一样，它能够存放各种类型的对象。可以简单的认为，向量是一个能够存放任意类型的动态数组。

由于vector是标准库中的类，在使用vector的时候我们需要包含标准库的头文件 < vector > .

3.1 vector 的初始化

vector <类型> 标识符（最大容量，初始所有值）

<1>第一个是空的整形vector，没有给他添加任何元素。

<2>第二个初始化了一个有3个元素的vector，由于并没有指定初始 值，将会使用编译器默认的初始值。

<3>第三个初始化了含有3个a的字符vector，括号中第二个值代表着所有元素的指定值。

<4>第四个vector通过拷贝vec3中的元素初始化vec4，它们的元素会一模一样。

下面附上一些常用到的函数

int size() const:返回向量中元素的个数

bool empty() const:判断向量是否为空，若为空，则向量中无元素

void pop_back():删除向量中最后一个元素

void push_back(const T& x):向量尾部增加一个元素X

iterator insert(iterator it,const T& x):向量中迭代器指向元素前增加一个元素x

void assign(int n,const T& x):设置向量中前n个元素的值为x
